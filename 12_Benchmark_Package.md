# HORB v1.0 — 12 Benchmark Package

This zip contains the full set of 12 HORB benchmark folders.

Each benchmark folder includes:

- `README.md`
- `00_Design_Brief.md`
- `UseCase_System_Prompt.md`
- `Strict_Grading_Guidelines.md`
- `Source_Documents_Manifest.md`
- `SourceDocs.zip`
- `source_docs/` with the individual fictional source documents

Package contents:

- Benchmark folders: 12
- Individual source documents: 336

All scenarios and documents are fictional and intended for AI evaluation / portfolio use only. They are not clinical decision support, medical advice, or patient-care guidance.

## GitHub note

Uploading this `.zip` file to GitHub will make the benchmark package downloadable, but GitHub will not automatically unpack it into browsable folders. For a prettier repository later, unzip this package on a computer and upload the `benchmarks/` folder itself.
