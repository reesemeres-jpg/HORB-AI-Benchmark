# HORB Roadmap

## Version 1.0

Initial public portfolio release.

Includes:

- 12 healthcare operational benchmark workflows
- 336 fictional operational source documents
- Benchmark summaries
- Methodology
- Capability taxonomy
- Failure-mode taxonomy
- Portfolio overview
- One-page project summary

## Planned Future Work

### 1. Public Sample Benchmark

Create one fully public sample benchmark that includes:

- System instructions
- User prompt
- Fictional source documents
- Expected reasoning path
- Grading rubric
- Example model failure modes

### 2. Scoring Sheet

Develop a standardized scoring sheet that reviewers can use to compare model outputs across benchmarks.

### 3. Model Testing

Test available AI models against selected HORB tasks when access is available.

### 4. Expanded Domains

Potential future benchmark domains:

- Utilization review
- Transfer center coordination
- Inpatient bed management
- Specialty referral triage
- Prior authorization review
- Clinical documentation integrity
- Nurse staffing and assignment coordination

### 5. GitHub Documentation Improvements

Add clearer documentation for:

- How to run a benchmark
- How to grade a response
- How to interpret model failures
- How to create a HORB-style benchmark variant

### 6. Version 2.0

Future version goals:

- More standardized scoring
- More diverse operational settings
- Broader non-hospital workflows
- Public sample tasks
- Structured reviewer worksheets
