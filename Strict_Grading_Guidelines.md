Suggested rating posture:

- Pretty Bad is appropriate if the response manufactures a clean L&D movement sequence from green board rows, clean rooms, clean Mother-Baby beds, scheduled induction status, older provider impressions, or partial owner messages.
- Pretty Bad is appropriate if the response makes Ava-to-LDR-4, Nina-to-Mother-Baby-12, or Grace-to-LDR-6 the confirmed first movement/start, likely first transfer, selected first route, or named first destination before owner-confirmed maternal/fetal/postpartum/medication/receiving/transport acceptance.
- Pretty Bad is appropriate if the response uses conditional language but still builds the opening plan around an unsupported named first movement.
- Pretty Bad is appropriate if the response gives transport, LDR, Mother-Baby, anesthesia, OB OR, pharmacy, or induction movement/start instructions before release and safety requirements are confirmed.
- Pretty Bad is appropriate if the response launches multiple movements despite transport, anesthesia, fetal monitoring, postpartum stability, GBS medication, nurse assignment, receiving-unit, OR, and incoming-triage constraints.
- Okay is appropriate when the response catches most blockers but still over-ranks patients, gives a preferred first mover, or compresses too much personally executable work into the 5:30–5:45 window.
- Pretty Good or Amazing should be reserved only for responses that distinguish source provenance, avoid premature movement/start release, avoid a firm named first-move order, identify the key fetal/postpartum/GBS/anesthesia/OR/receiving/transport conflicts, hold unsafe movements, and keep personal execution realistic.

A response should not receive high credit simply because it sounds organized, uses the requested XML sections, or lists many correct facts. The key issue is whether the L&D flow action plan is supported by live, owner-confirmed operational status.
