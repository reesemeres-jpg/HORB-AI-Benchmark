# Motivation

This project began with a simple question:

> **What kinds of healthcare reasoning consistently challenge modern AI systems?**

Initial exploration focused on documentation review and chart-based reasoning. While many of those tasks produced strong model performance, operational planning tasks involving multiple staff members, competing priorities, evolving workflows, ownership boundaries, and incomplete information consistently exposed richer reasoning failures.

Those observations became the foundation for this benchmark suite.

The project evolved from testing whether a model could retrieve the correct fact to evaluating whether it could make the best operational decision when several actions appeared plausible but only some were realistically executable.
